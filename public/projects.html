Football Results Automation Tool

Problem

Manual process required 1-2 hours of work every week.

Whilst compiling the football club's weekly newsetter for the roughly 2000 subscribers, time-consuming manual information collation and styling was required. This involved finding and presenting information relevant to match results, game times, oppositions, locations, dates etc... for each of the 9 teams.

Solution

Reduces time to complete task from 1-2 hours down do approximately 1 minute. Greatly improves visual display.

The Football Results Automation Tool is a web application uses Google's Puppeteer API alongside Headless Chrome to automatically find and extract all information relevant to the club's weekly results.
It also provides a rich display which is directly portable into the MailChimp emailing service.

Radar Calibration Tool

Problem

Error-prone, labour-intensive and unscalable calibration process required a day to complete.

At CEA Technologies, the previous process for calibrating phased-array radars involved using a set of MATLAB scripts to process experimental data. These scripts were required to be ammended for each different variation of calibration which resulted highly error-prone and caused rapid code degradation. The process required a technician to email the experimental results to an engineer who would process the results using the MATLAB scripts and return to the technician for troubleshooting, taking approximately 1 day in total.

Solution

Completely eliminated the requirement of an engineer in the calibration process and reduced calibration times from a day to just minutes.

The Radar Calibration Tool provided a comprehensive graphical user interface used to configure calibration parameters, eliminating the need to modify code for each calibration.
The tool also provided various data visualisations of the calibration result including 3D spatial plotting and heat maps. These allowing technicians to quickly and more easily understand and diagnose problems.

Money Tracker

Problem

No practical way to see how money is being spent or trends over time from multiple accounts.

Bank statements are largely unhelpful, presenting every transaction individually without any ability to analyse trends over time, group related transactions or maintain more meaningful descriptions. This problem is amplified when finances are distributed across multiple accounts.

Solution

Indexes and organises every transaction across multiple accounts and provides insights to relative spending and trends over time.

Money Tracker takes bank statement in PDF format and extracts every transaction into an SQL database. Transactions can then be categorised into a set of hierarchial tags. This then allows multiple data visualisations to be generated, including pie charts of comparitive spending and graphs of weekly/monthly/yearly spending trends in any category.
